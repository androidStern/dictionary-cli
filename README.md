##  English Dictionary CLI
> English Dictionary from the command line 

## Install
```
$ npm install -g english-dictionary 
```

## Usage
```
$ definition <word> [options]
```
![dictionary cli demo](dictionary-cli-demo.gif)

## Example
```
$ defintion abate
```
Or use the short version `def`
```
$ def abate
```
